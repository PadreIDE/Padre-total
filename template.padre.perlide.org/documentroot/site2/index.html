Padre, the Perl IDE

Padre is a Perl IDE, an integrated development environment, a simple yet powerful text editor written in Perl.

Our primary focus is to create a environment for learning and development, with an extensible plug-in system to support additional functionality.

Learn more »

Features

  • Customizable syntax highlighting
  • Context sensitive help
  • Code completion
  • Refactoring tools
  • Multi-platform: runs on Windows, Linux, Mac OS X
  • Free and Open Source
  • Written in Perl
