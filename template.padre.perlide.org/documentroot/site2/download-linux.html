Linux

All Linux distributions come with Perl, and the most important ones already have a Padre package. So, choose your flavor:

Ubuntu

apt-get install padre

Debian

aptitude install padre

Fedora

yum install perl-Padre

Mandriva

urpmi perl-Padre

Other Linux and *BSD

For installation on other Linux distributions and *BSD, please look at the detailed installation instructions.
