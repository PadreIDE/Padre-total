Perl IDE

There are several cross platform IDEs for Perl.

Eclipse
with the EPIC plugin or with Perlipse using dltk
Kephra
Programmers Editor along perl-like Paradigms, written in Perl
ActiveState
proprietary Komodo IDE or free Komodo Edit
Padre
The Perl Application Development and Refactoring Environment (written in Perl 5)
Gvim
with Perl support
Emacs
with CPerl Mode
