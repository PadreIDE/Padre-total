Was ist Padre?

Padre ist nicht nur eine Perl - Entwicklungsumgebung, sondern auch ein Perl - Projekt.
Padre ist komplett in Perl geschrieben und unterstützt die Entwicklung von Projekten in Perl 5, Perl 6 und diversen weiteren Programmiersprachen.

Padre - Installation unter Windows

Diese Anleitung benötigt kein vorinstalliertes Perl, das Installationspaket enthält alle notwendigen Komponenten.

  1. Padre von der offiziellen Padre Download-Seite herunterladen. Diese Anleitung basiert auf dem Padre Standalone - Paket in der Version 0.45, sie sollte aber auch mit jeder späteren Version funktionieren.
  2. Die heruntergeladene .msi - Datei doppelt anklicken.
  3. Sofern die Nutzungs- und Lizenzbedingungen akzeptiert wurden, reicht ein Klick auf den Install - Button, um die Installation zu starten.
  4. Padre kann jetzt via Start, Programme, Padre Standalone und Padre gestartet werden.
  5. Das CPAN - Perl-Archiv verfügt möglicherweise über eine neuere Version als das aktuellste Installationspaket, ein Update ist also zu empfehlen.

Padre - Update unter Windows

Padre wird ständig weiterentwickelt, durchschnittlich alle zwei Wochen, je nach Umfang der änderungen aber teilweise auch wöchentlich erscheinen neue Versionen. Ein Update aktualisiert Padre auf die neuste stabile Version und ist i.d.R. zu empfehlen:

  1. Auf Start klicken
  2. Auf Ausführen klicken (entfällt ab Vista)
  3. (Den Standardpfad vorausgesetzt) folgende Zeile eingeben oder reinkopieren: C:\strawberry\perl\bin\perl.exe -MCPAN -e "CPAN->upgrade('/^Padre/')"
  4. Der Updateprozess kann bis zu 70 Minuten dauern (abhängig von der Anzahl der notwendigen Updates und der Geschwindigkeit der Internetverbindung).
  5. Sollte das Update nicht starten, den o.g. Vorgang mit folgender Zeile wiederholen: perl.exe -MCPAN -e "CPAN->upgrade('/^Padre/')"
