You really need to start with a Bootstrap installation, because at the end of this, this will install into c:\strawberry.

1) Create a par file for Alien::wxWidgets, because we need to install a patched version.

cd %USERPROFILE%
wget $CPAN/M/MB/MBSRBON/Alien-wxWidgets-0.43.tar.gz
wget (patchfile)
wget (mtee)
gzip -d Alien-wxWidgets-0.43.tar.gz
tar -xf Alien-wxWidgets-0.43.tar
./Alien-wxWidgets-0.43/share/bin/patch (patchfile) 
cd Alien-wxWidgets-0.43
perl Makefile.PL
perl -MPAR::Dist -eblib_to_par
(wait 40 minutes)
copy Alien-*.par c:\

2) Install Perl::Dist::Strawberry from svn

svn co http://svn.ali.as/cpan/trunk/Perl-Dist-Strawberry
cd Perl-Dist-Strawberry
perl Makefile.PL && dmake && dmake test
dmake install
cd ..

3) Install Perl::Dist::Padre from svn

svn co http://padre.perlide.com/trunk/Perl-Dist-Padre
cd Perl-Dist-Padre
perl Makefile.PL && dmake && dmake test
dmake install
cd ..

4) Actually build the dist...

set ...=C:\...
perldist Padre 2>&1 | mtee padre-install.txt
(wait for 3 hours)
