Padre on Strawberry Perl
Release notes

   Unzip the file
   put it in some place without a path in it. e.g c:\strawberry
   Run the relocation.pl.bat
   Run the update_env.pl.bat
   Open the Control Panel / System / Advanced / Environment Variables / OK / OK
       This will make the previous changes work *without* a reboot.
   Run the setup_padre.pl.bat
       This will put an icon of Padre on your desktop and
	   you can run it by double clicking on the icon
